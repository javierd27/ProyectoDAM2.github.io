Acerca de Hotelia

Hotelia v1.0

Sistema de Gestión Hotelera Integral

Plataforma de escritorio desarrollada en Java Swing

Descripción

Hotelia es un sistema integral diseñado para simplificar la gestión de hoteles de todos los tamaños.

Características

  • ✓ Gestión completa de clientes
  • ✓ Sistema de reservas en tiempo real
  • ✓ Control de habitaciones
  • ✓ Gestión de empleados y roles
  • ✓ Facturación automática
  • ✓ Servicios adicionales
  • ✓ Reportes y estadísticas

Información Técnica

Versión: 1.0
Plataforma: Java 8+
Interfaz: Swing
Base de Datos: MySQL
