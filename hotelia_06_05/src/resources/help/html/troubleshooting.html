Solución de Problemas

No puedo iniciar sesión

Verifique que está usando el nombre de usuario y contraseña correctos. Si olvidó su contraseña, contacte al administrador.

La aplicación se ejecuta lentamente

Cierre otras aplicaciones, verifique la conexión a la base de datos y reinicie la aplicación.

No puedo guardar cambios

Asegúrese de que la base de datos esté disponible y que tiene permisos para editar.

Las habitaciones no se muestran correctamente

Recargue la vista o reinicie la aplicación. Verifique que las habitaciones estén activas.

Error de conexión a base de datos

Verifique la conexión de red y las credenciales. Contacte al administrador de sistemas.
